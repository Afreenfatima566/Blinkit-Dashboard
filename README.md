1. Project Title / Headline

Blinkit Sales Performance & Profitability Dashboard
An interactive Power BI dashboard designed to analyze FMCG sales performance, outlet efficiency, product categories, and customer ratings across different outlet types, sizes, and locations.

2. Short Description / Purpose

The Blinkit Sales Performance Dashboard provides a comprehensive view of sales, profitability, item distribution, and customer satisfaction. It enables stakeholders to evaluate outlet performance, understand product-level trends, and make data-driven decisions to improve revenue and operational efficiency.

Source: FMCG retail sales dataset (Blinkit-inspired dataset)
The dataset contains transactional and master data related to:
Outlet details (location tier, size, type, establishment year)
Item attributes
Sales values, profit margins, and item visibility
Customer ratings
This structured dataset enables multi-dimensional analysis across outlets, products, and time.
